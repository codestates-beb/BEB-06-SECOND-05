# ⚙️ How To Use
>### git Clone
```bash
# Clone this repository
$ git clone git@github.com:codestates-beb/BEB-06-SECOND-05.git
````
>### .env setting
```bash
DATABASE_PASSWORD=""
DATABASE_IP=""
INFURA_KEY=""
SECRET_KEY=""
SERVER_SECRET_KEY=""
```
>### Client
```bash
$ npm install --legacy-peer-deps
$ npm start
```
>### Server
```bash
$ npm install
$ npm start
```
